# Task Manager App

This is a full-stack Task Manager application built with:

- **Backend:** Django
- **Frontend:** React
- **Features:** User authentication, task CRUD, task due dates, and task completion status.

## How to Run

### Backend (Django)
1. Create virtual environment: `python -m venv venv`
2. Activate it: `source venv/bin/activate` (Linux/macOS) or `venv\Scripts\activate` (Windows)
3. Install dependencies: `pip install -r requirements.txt`
4. Run server: `python manage.py runserver`

### Frontend (React)
1. Navigate to `frontend/`
2. Install dependencies: `npm install`
3. Run development server: `npm start`
